## 🔍 TROUBLESHOOTING

### Issue: 404 NOT_FOUND Error

**Symptom:** All routes return 404  
**Solution:**
1. Verify `vercel.json` is in project root
2. Check Root Directory setting is blank
3. Redeploy with clear build cache

### Issue: Backend Source Code Displayed

**Symptom:** Browser shows raw JavaScript code  
**Solution:**
1. Ensure `module.exports = app` at end of index.js
2. Check routing in vercel.json matches folder structure
3. Verify no nested vercel.json files in subdirectories

### Issue: CORS Errors

**Symptom:** Frontend can't access API  
**Solution:**
1. Check CORS configuration in backend index.js includes:
   - `https://devproflow.com`
   - `https://www.devproflow.com`
   - `https://api.devproflow.com`
2. Ensure API client uses correct base URL
