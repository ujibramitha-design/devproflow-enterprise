# KPRFlow Enterprise - Web Application

## 🎯 Overview
Complete full-stack web application for KPRFlow Enterprise with DevPro-Enterprise functionality.

## 📊 Architecture
- **Frontend**: Next.js 15.1.11 + React 19.0.0 + TypeScript
- **Backend**: Node.js + Express.js + Supabase
- **Database**: Supabase PostgreSQL + Firebase Real-time Database
- **Integration**: WhatsApp Business API + Multi-platform support

## 🚀 Features
- ✅ Real-time Firebase integration
- ✅ WhatsApp Business API
- ✅ Complete dashboard system
- ✅ Customer-facing portal
- ✅ Executive dashboards (BOD, Finance)
- ✅ Unit catalog management
- ✅ Multi-platform deployment (Electron, Capacitor)
- ✅ DevPro-Enterprise branding

## 📂 Structure
```
02_Web_Application/
├── 📂 frontend/ (Complete Next.js app)
│   ├── 📂 src/app/ (Pages and layouts)
│   ├── 📂 src/components/ (60+ UI components)
│   ├── 📂 src/lib/ (Firebase, WhatsApp, utilities)
│   └── 📂 [Multi-platform configs]
├── 📂 backend/ (Express API + Supabase)
│   ├── 📄 server.js (Express server)
│   ├── 📂 src/ (API routes)
│   └── 📂 [Database connections]
├── 📂 docs/ (Documentation)
└── 📂 tests/ (Test files)
```

## 🛠️ Development
```bash
# Frontend
cd frontend
npm install
npm run dev

# Backend
cd backend
npm install
npm run dev
```
